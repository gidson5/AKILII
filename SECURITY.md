# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Akili, please **do not open a public GitHub issue**.

Email us at **akilihq12@gmail.com** with:
- A description of the vulnerability
- Steps to reproduce
- Potential impact

We will respond within 48 hours and aim to release a fix within 7 days for critical issues.

## Scope

- Smart contract vulnerabilities (PolicyRouter at `0x0c0345b3DFBFFD4300255F9DB933A4751cCD2124`)
- API endpoint security issues
- Wallet data exposure or leakage
- x402 payment verification bypasses

## Out of Scope

- Issues in third-party dependencies (report to them directly)
- Social engineering attacks
- Theoretical vulnerabilities without a proof of concept
